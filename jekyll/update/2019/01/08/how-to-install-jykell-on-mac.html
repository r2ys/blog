How to install jykell on mac!

Jan 8, 2019

如何在mac上安装jykell

1.安装rvm

curl -L get.rvm.io | bash -s stable

source ~/.rvm/scripts/rvm

rvm -v

2.安装ruby

rvm install 2.6.0

rvm list

3.安装jykell

gem source

gem install jekyll

jekyll -version

4.安装bundle

gem install bundle

运行bundle，安装Gemfile里的插件

bundle install

5.初始化web项目

jekyll new r2ys-site

cd r2ys-site/

6.本地localhost:4000测试

jekyll serve
jekyll serve --watch --baseurl '/blog'

7.内容生成

jekyll build
